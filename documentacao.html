nesta seçao vamos apresentar tudos os comandos utilizados pelo processador p3com uma breve discrições e os flags que sáo afetados pelos comandos

comandos sem argumentos

NOP → apenas aumenta um cilo de relogio

ENI → ativa interrupções (E)

DSI → desativa interrupções (E)

STC → coloca Carry = 1 (C)

CLC → coloca Carry = 0 (C)

CMC → inverte Carry (C)

RET → retorna de subrotina

RTI → retorna de interrupção (E Z C N O)


comandos com constante

INT (constante) → gera interrupção n (E Z C N O – limpa)

RETN (consante) → retorna e limpa n valores da pilha


comandos com 1 argumento

NEG (registo / memória)→ muda sinal (Z C N O)

INC (registo / memória) → incrementa (Z C N O)

DEC (registo / memória)→ decrementa (Z C N O)

COM (registo / memória)→ NOT (Z N)

PUSH (registo / memória / constante)→ mete na pilha

POP (registo / memória)→ retira da pilha


shifts e rotações

SHR (registo/memória), (constante)→ shift direita (Z C N)

SHL (registo/memória), (constante) → shift esquerda (Z C N)

SHRA (registo/memória), (constante)→ shift direita com sinal (Z C N O)

SHLA (registo/memória), (constante)→ shift esquerda com overflow (Z C N O)

ROR (registo/memória), (constante)→ roda direita (Z C N)

ROL (registo/memória), (constante)→ roda esquerda (Z C N)

RORC (registo/memória), (constante)→ roda direita com carry (Z C N)


comandos com 2 argumentos

Aritméticas e lógicas

ADD (registo / memória), (registo/memória/constante)→ soma (Z C N O)

ADDC (registo / memória), (registo/memória/constante) → soma com carry (Z C N O)

SUB (registo / memória), (registo/memória/constante) → subtrai (Z C N O)

SUBB(registo / memória), (registo/memória/constante) → subtrai com borrow (Z C N O)

MUL (registo ), (registo/memória)→ multiplica (Z)

DIV (registo ), (registo/memória)→ divide (Z O)

CMP (registo/memória), (registo/memória/constante)→ compara (Z C N O)

TEST (registo/memória), (registo/memória/constante)→ AND só para flags (Z N)

AND (registo), (registo/memória/constante)→ E lógico (Z N)

OR(registo), (registo/memória/constante) → OU lógico (Z N)

XOR (registo), (registo/memória/constante)→ OU exclusivo (Z N)

MOV (registo/memória/SP), (registo/memória/constante/SP)→ copia valor

MVBL (registo), (registo/memória/constante)→ move byte baixo

XCH (registo), (registo/memória/constante)→ troca valores


saltos

JMP (label/registo/memória/PC)→ salto absoluto

CALL (label/registo/memória/PC) → chama subrotina

BR (constante/label offset)→ salto relativo

JMP.cond (label/registo/memória/PC)→ salta se condição

CALL.cond (label/registo/memória/PC)→ chama se condição

br.cond (constante/label offset) → salto relativo condicional

pseudo-instruções

ORIG (constante)→ define endereço

EQU (constante)→ define constante

WORD (constante)→ guarda valor

STR (string)→ guarda string

TAB (constante)→ reserva espaço




além dos comandos existe tambem as condiçoesmencionadas nos comnados em cima como cond, sendo estas aplicadas sobre as flags

condições

Z / NZ → zero → (Z)

C / NC → carry → (C)

N / NN → negativo → (N)

O / NO → overflow → (O)

P / NP → positivo → (Z,N)

I / NI → interrupções → (E)
